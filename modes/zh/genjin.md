# 跟进追踪 (Follow-up Tracking)
#
# Chinese-language follow-up mode for tracking application follow-ups.
# Corresponds to the English `followup.md` mode.
#
# Usage: /career-ops followup (when language.modes_dir is set to modes/zh)

## 跟进流程

### 1. 检查申请状态
从 `data/applications.md` 中读取所有申请记录，识别以下状态：
- `applied` — 已投递，等待回复
- `responded` — 已收到回复
- `interview` — 面试阶段
- `offer` — 收到 offer
- `rejected` — 被拒绝
- `discarded` — 主动放弃

### 2. 识别需跟进的申请

**跟进规则**：
| 状态 | 跟进时机 | 说明 |
|------|----------|------|
| 已投递3-5天 | 第一次跟进 | 确认简历已收到 |
| 已投递7-10天 | 第二次跟进 | 询问进度 |
| 面试后2-3天 | 感谢跟进 | 感谢面试机会 |
| Offer 后 | 薪资谈判 | 讨论薪资福利 |

### 3. 生成跟进消息

**BOSS直聘跟进模板**：
```
您好，我是[姓名]，[日期]投递了贵公司的[职位]。想确认一下简历是否收到，以及后续的面试安排。期待您的回复，谢谢！
```

**邮件跟进模板**：
```
主题：[职位] 申请跟进 - [姓名]

尊敬的 HR：

您好！

我是[姓名]，于[日期]通过[平台]投递了贵公司的[职位名称]职位。

我对这个职位非常感兴趣，想确认一下我的申请是否已经收到，以及后续的面试安排。

如果需要补充任何材料，请随时告知。

期待您的回复，谢谢！

此致
敬礼
[姓名]
[电话]
[邮箱]
```

### 4. 更新跟进记录

将跟进记录写入 `data/follow-ups.md`：

```markdown
## 跟进记录

| 日期 | 公司 | 职位 | 跟进类型 | 状态 | 备注 |
|------|------|------|----------|------|------|
| 2025-01-15 | 字节跳动 | 高级前端工程师 | 第一次跟进 | 已发送 | 通过BOSS直聘发送 |
| 2025-01-20 | 字节跳动 | 高级前端工程师 | 第二次跟进 | 已回复 | 约了下周面试 |
```

### 5. 生成跟进提醒

识别即将到期的跟进：
- 明天需要跟进的申请
- 本周需要跟进的申请
- 已超过跟进窗口的申请（标记为"可能已无回复"）

## 中国市场特殊跟进规则

### BOSS直聘跟进
- 通过 BOSS 直聘 App 内的聊天功能跟进
- 避免频繁发送消息（每天最多1次）
- 如果对方已读不回，等待2-3天再跟进

### 智联招聘/前程无忧跟进
- 通过平台站内信跟进
- 注意平台的消息频率限制

### 邮件跟进
- 外企通常更接受邮件跟进
- 国内公司可能更倾向于平台内消息
- 邮件主题要简洁明了

### 跟进禁忌
- ❌ 每天都发消息
- ❌ 在非工作时间发送
- ❌ 语气过于急切
- ❌ 同时向同一家公司投递多个职位后频繁跟进
