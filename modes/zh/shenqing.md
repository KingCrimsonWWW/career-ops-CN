# 申请助手 (Application Assistant)
#
# Chinese-language application mode for filling out job application forms.
# Corresponds to the English `apply.md` mode.
#
# Usage: /career-ops apply (when language.modes_dir is set to modes/zh)

## 申请流程

### 1. 读取职位信息
从用户提供的 URL 或文本中提取：
- 公司名称
- 职位名称
- 职位要求
- 申请平台（BOSS直聘、智联招聘、前程无忧、猎聘等）

### 2. 加载用户信息
从 `config/profile.yml` 和 `modes/_profile.md` 中读取：
- 基本信息（姓名、联系方式）
- 工作经历
- 教育背景
- 技能清单
- 项目经验

### 3. 生成申请材料

#### 自我介绍（中文）
```
面试官您好，我是[姓名]，[X]年[X]领域工作经验。

在过去的工作中，我主要负责[核心职责]，取得了[具体成果]。

我对贵公司的[职位名称]非常感兴趣，因为[原因]。

我相信我的[相关经验/技能]能够为团队带来价值。
```

#### 职位匹配分析
| 职位要求 | 我的经验 | 匹配度 |
|----------|----------|--------|
| 要求1 | 经验1 | ✅ 完全匹配 |
| 要求2 | 经验2 | ⚠️ 部分匹配 |
| 要求3 | 经验3 | ❌ 需要学习 |

#### 项目经验描述（STAR 法则）
- **Situation（背景）**: 项目背景和挑战
- **Task（任务）**: 我的职责
- **Action（行动）**: 采取的措施
- **Result（结果）**: 取得的成果（量化）

### 4. 优化简历内容

针对特定职位优化简历：
- 突出相关经验
- 调整技能顺序
- 量化成果（如"提升性能30%"、"用户增长200%"）
- 使用行业术语

### 5. 填写申请表单

如果使用 Playwright 辅助填写：
1. 识别表单字段
2. 映射到用户信息
3. 生成填写内容
4. 提示用户确认后提交

## 中国市场特殊要求

### 简历格式
- 中文简历通常1-2页
- 附上照片（部分公司要求）
- 注明期望薪资
- 注明到岗时间

### 常见问题及回答模板

**Q: 为什么离开上一家公司？**
```
我非常感谢上一家公司给予我的成长机会。离开主要是因为：
1. 希望在[技术/业务]领域有更深入的发展
2. 对贵公司的[具体方面]非常感兴趣
3. 希望在更大的平台上发挥自己的价值
```

**Q: 你的期望薪资是多少？**
```
根据我的经验和市场行情，我期望的薪资范围是[XX-XX]K/月（年包[XX-XX]万）。
当然，我也会综合考虑公司的整体福利和发展机会。
```

**Q: 你有什么问题想问我们？**
```
1. 这个职位的具体工作内容和团队架构是怎样的？
2. 公司对这个职位的期望和考核标准是什么？
3. 团队的技术栈和工作流程是怎样的？
4. 公司对员工的培养和晋升机制是怎样的？
```

## BOSS直聘申请技巧

1. **开场白要简洁**
   - 不要只说"你好"
   - 简要介绍自己和求职意向

2. **突出亮点**
   - 1-2个最相关的经验或技能
   - 量化成果

3. **表达诚意**
   - 说明为什么对这个职位感兴趣
   - 展现对公司的了解

4. **示例开场白**
   ```
   您好！我是[姓名]，[X]年[X]经验，曾在[公司]负责[项目]，
   取得了[成果]。对贵公司的[职位]非常感兴趣，
   期待有机会进一步沟通！
   ```
