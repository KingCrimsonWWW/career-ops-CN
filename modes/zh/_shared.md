## 评分系统 (Scoring System)

### A-F + G 评估体系

每个职位评估包含 7 个评估块：

| 块 | 名称 | 说明 |
|----|------|------|
| A | 技术匹配 | 技术栈、架构经验、业务领域匹配度 |
| B | 职业发展 | 晋升空间、技术深度、管理机会 |
| C | 薪资福利 | 月薪、年包、股票期权、公积金、年终奖 |
| D | 公司质量 | 规模、稳定性、行业地位、技术影响力 |
| E | 工作生活 | 加班文化、弹性工作、带薪假期 |
| F | 地理位置 | 城市等级、通勤时间、远程办公 |
| G | 综合评估 | 职位真实性、整体直觉、风险评估 |

### 评分标准

每块评分 1-5：
- ⭐ 1/5: 严重不匹配
- ⭐⭐ 2/5: 部分匹配，有明显不足
- ⭐⭐⭐ 3/5: 基本匹配
- ⭐⭐⭐⭐ 4/5: 良好匹配
- ⭐⭐⭐⭐⭐ 5/5: 完美匹配

### 综合评分计算

综合评分 = (A×0.20 + B×0.20 + C×0.20 + D×0.15 + E×0.15 + F×0.05 + G×0.05)

评分阈值：
- 4.0+ → 强烈推荐申请
- 3.5-3.9 → 建议申请
- 3.0-3.4 → 可以考虑
- 2.5-2.9 → 谨慎考虑
- <2.5 → 不推荐

## 中国市场维度 (China Market Dimensions)

### 公司类型分类

| 类型 | 说明 | 典型特征 |
|------|------|----------|
| 大厂 | 字节跳动、阿里、腾讯、美团、京东、百度、网易、快手、小米 | 高薪、高强度、技术挑战 |
| 独角兽 | 小红书、滴滴、B站、拼多多、SHEIN | 高薪、快速增长、期权机会 |
| 外企 | Google、Microsoft、Apple、Amazon 中国区 | 工作生活平衡、国际视野 |
| 国企/央企 | 中国银行、中国移动、华为（部分） | 稳定、福利好、节奏较慢 |
| 创业公司 | 早期融资 | 高风险高回报、股权激励 |

### 薪资结构

中国科技行业薪资通常包括：
- 基本月薪（税前）× 月数（12-16个月）
- 年终奖（通常1-6个月）
- 签字费（入职奖金，部分公司）
- 股票期权（通常4年 vesting）
- 五险一金（养老、医疗、失业、工伤、生育 + 住房公积金）

### 工作文化指标

**红旗**：
- 996（早9晚9，每周6天）
- 大小周（交替5天/6天工作周）
- "能接受高强度工作"
- "创业心态"
- 频繁招聘同一职位（高离职率信号）

**绿旗**：
- 弹性工作时间
- 远程办公选项
- 明确的带薪年假天数
- 不鼓励加班
- 员工评价正面

## 全局规则 (Global Rules)

1. 所有评估报告使用中文撰写
2. 薪资以人民币（RMB）为单位
3. 薪资范围标注为"月薪·X薪"格式（如"30-50K·16薪"）
4. 公司名称优先使用中文（如有）
5. 技术术语可使用英文（如 React, TypeScript）
6. 评估时考虑中国市场特有因素（公积金比例、年终奖、股票期权）
